• El elemento style debe colocarse en el elemento de tipo head y contiene el código CSS3 que define la apariencia de la página.

Ejemplo

<!DOCTYPE html>
<html lang="es">
<head>
 <meta charset="UTF-8">
 <meta name="viewport"
   content="width=device-width">
 <title>CSS</title>
 <style>
  h1 {
   color: yellow;
   background-color: blue;
  }

  p {
   color: white;
   background-color: black;
  }
 </style>
</head>
<body>
 ...
</body>
</html>
